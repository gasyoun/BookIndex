# BookIndex (Zalizniakiada)

**BookIndex** — автономный интерактивный веб-справочник, корпусная лаборатория и интеллектуальная база знаний по научному наследию **А. А. Зализняка**. 

Проект эволюционировал из простого индекса в полноценную исследовательскую платформу v17.6, объединяющую модульную архитектуру, интеллектуальный поиск с ИИ-ассистентом, систему геймификации и инструменты цифровых гуманитарных наук (Digital Humanities).

- **Демо**: [gasyoun.github.io/BookIndex/index.html](https://gasyoun.github.io/BookIndex/index.html)
- **Статус**: v17.6 «Secret Expedition» (обновлено 2026-05-06).
- **Руководство для детей**: [KIDS_GUIDE_RU.md](KIDS_GUIDE_RU.md)
- **Лицензия**: [Apache License 2.0](LICENSE)

## Ключевые возможности v17.6

### 🧠 Интеллектуальный поиск и AI Copilot
*   **Morphological Search**: Поиск со стеммингом, учитывающий падежи и окончания русского языка.
*   **AI Insights**: Динамические подсказки в карточках, выявляющие скрытые связи между терминами, именами и топонимами.
*   **Cross-Corpus Linking**: Автоматическая связь между видеолекциями (191 лекция) и текстом корпуса.

### 🏛️ Инструментарий Digital Humanities
*   **Topic Clustering**: Автоматическое группирование тысяч сущностей в концептуальные кластеры («Дальнее чтение»).
*   **Network Analysis**: Математический расчет «Хабов знаний» и визуализация сетевой центральности.
*   **Semantic Web**: Полная поддержка JSON-LD (Schema.org) для интеграции в мировой научный контекст.

### 🏆 Лингвистическая геймификация (Linguistic Expedition)
*   **Interactive Quiz**: Четырехуровневый тренажер лингвиста (от этимологии до акцентологии).
*   **Achievement System**: Система значков за научные открытия и исследование корпуса.
*   **Secret Achievements**: «Пасхалки» и скрытые награды для самых внимательных исследователей.

### 📝 Дневник исследователя 2.0
*   **Persistent Storage**: Переход на **IndexedDB** для хранения неограниченного объема заметок.
*   **Markdown Export**: Выгрузка результатов исследования в профессиональный формат.
