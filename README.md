# BookIndex (Zalizniakiada)

**BookIndex** — автономный интерактивный веб-справочник, корпусная лаборатория и мультимедийный архив по научному наследию **А. А. Зализняка**. 

Проект вырос из индекса одной книги в глобальную базу знаний, объединяющую текстовый корпус («Из жизни слов и языков»), архив видеолекций и инструменты для глубокого лингвистического исследования в любых условиях.

- **Демо**: [gasyoun.github.io/BookIndex/index.html](https://gasyoun.github.io/BookIndex/index.html)
- **Статус**: v11.2 «Expeditionary & Semantic» (обновлено 2026-05-06).
- **Лицензия**: [Apache License 2.0](LICENSE)

## Ключевые возможности v11.2

### 📡 Режим экспедиции (Expeditionary Mode)
*   **100% Offline**: Полное кэширование корпуса и медиа-метаданных (PWA). Работает в архивах и полях без интернета.
*   **Mobile Optimized**: Адаптивный интерфейс с поддержкой жестов (свайпы), крупными touch-зонами и фоновым поиском (Web Workers).

### 🎬 Семантический видеоархив
*   **191 лекция**: Полный каталог видео с обложками и метаданными (236+ часов контента).
*   **Semantic Linking**: 812 интеллектуальных связей между видео и терминами корпуса (Deep Stemming Match).

### 📝 Дневник исследователя
*   **Аннотации**: Персональные заметки на карточках сущностей с автосохранением.
*   **Досье**: Формирование тематических подборок для научных проектов.
*   **Экспорт**: Выгрузка результатов в профессиональный Markdown.
