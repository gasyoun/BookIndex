# BookIndex Svelte Pilot

Изолированный прототип декларативного UI для постепенной миграции с vanilla runtime.

## Цель

- проверить, как удобно выражать текущие паттерны фильтрации/поиска в Svelte;
- не затрагивать основной артефакт `aaz-index.html` и прод-сборку;
- подготовить путь к возможной миграции на декларативные компоненты.

## Запуск

```bash
cd experimental/svelte-pilot
npm install
npm run dev
```

## Что внутри

- `src/App.svelte` — демонстрационный список сущностей с фильтрацией;
- `src/main.js` — bootstrap Svelte-приложения;
- `vite.config.mjs` — dev/build-конфигурация пилота.

## Ограничения

- это sandbox, не интегрирован в основной runtime;
- не участвует в `scripts/build_aaz_index.py`;
- нужен только как база для итераций по пункту «декларативный UI».
